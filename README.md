# Спецификация
- Стандарт С++20
-  CMake 3.15
# Инструкция по сборке

## Откройте терминал (в macOS) или Командную строку / Git Bash (в Windows) и выполните команды:
1. Клонируйте проект
   `git clone git@github.com:p0sd7/NauKernelTest.git`
2. Перейдите в папку проекта
   `cd path`
3. Создайте папку build и перейдите в нее
   `mkdir build`
   `cd build`
4. Сконфигурируйте проект
   `cmake ..`
5. Скомпилируйте проект
   `cmake --build . --config Release`

После успешной сборки исполняемый файл появится в папке build
-----
### P.S.
Я выбрал std::unique_ptr для хранения данных, т.к. он более предсказуемый и безопасный.
unique_ptr гарантирует, что компонентами владеет только сцена. При использовании shared_ptr выше риск появления утечки памяти,
в случае если какой-то объект владеющий shared_ptr'ом не удалился. И unique_ptr меньше весит.
